Build instructions:
Originaly built in Ubuntu Linux
(1) open the Terminal
(2) navigate to the root directory containing CMakeLists.txt
(3) run the command: cmake -Bbuild
(4) navigate to the build directory that was created
(5) run the command: make
(6) to run the program, run the command: ./CPSC453A2


Controls:
Up arrow key: decrease the number of subdivisions by 1
Down arrow key: increase the number of subdivisions by 1
Right arrow key: go to next fractal type
Left arrow key: go to previous fractal type

GUI also provided
Drag the UI sub window out of the way if you do not want it
UI buttons with fractal names allow switching to selected fractals
Slider controlls the number of subdivision levels
